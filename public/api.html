Maria's Rest APIs

The following are the routes:

  • /flowers to get all flowers in a JSON document from my MongoDB
  • /updateflowernotes to post flower notes in my MongoDB
  • /redislistsample to get a list to my RedisDB
  • /redissetsample to get a set to my RedisDB
  • /redishashsample to get a hash table to my RedisDB
  • /rediszsetsample to get a zset to my RedisDB
  • /putstringtoredis to put strings to my Redis DB
