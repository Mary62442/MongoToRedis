Maria's Rest APIs

The following are the routes:

  • /flowers to get all flowers in a JSON document from my MongoDB
  • /updateflowernotes to post flower notes in my MongoDB
  • /putlisttoredis to put a list to my RedisDB
  • /putsettoredis to put a set to my RedisDB
  • /puthashtoredis to put a hash table to my RedisDB
  • /putzsettoredis to put a zset to my RedisDB
  • /putstringtoredis to put strings to my Redis DB
